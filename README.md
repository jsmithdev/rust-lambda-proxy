# rust-lambda-proxy

A reverse proxy in Rust for Lambda

## Usage

See [lambda_runtime](https://crates.io/crates/lambda_runtime) for more details.

Watch the project

```bash
cargo lambda watch
```

Build the project

```bash
cargo lambda build --release
```
